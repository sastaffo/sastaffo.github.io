GitUser.py:

GitUser takes in a password and a username of user X. If user X has a location set on github,
the program will find all of X's followers who also have a location set and create a list of tuples
mapping their location to X's and vice versa.
The program interacts with the github API in multiple of these functions, getting the user's location and list of followers.

Helper.py:

helper.py contains 3 classes designed to help with gituser's functionality.
